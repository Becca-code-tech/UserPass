ctrl+shift+f ით მოძებნე BekaFix და იპოვი რა გავასწორე შენს კოდში.
ctrl+shift+f ით მოძებნე BekaAdd და იპოვი რა უნდა დაამატო.


Bekafix ის გარდა რა გავაკეთე: 
1) admin-home კომპონენტი დავამატე.
2) guard ები და სერვისები ფოლდერებში გადავახარისხე.
3) დავამატე models folder და posts.model.ts => მეოთხე დავალებაში გაიგებ მიზეზს

რა უნდა გააკეთო:
1)
  admin-page ს გადაარქვი სახელი. დაარქვი admin-login
  (admin.module შიც უნდა შეცვალო იპორტები. 
  შეიძლება აურიოს და მოგიწიოს ანგულარის სერვერის გათიშვა და თავიდან ng serve);

2) 
  home component გაალამაზე. დაამატე რამე ნავბარი 
  (ნავბარი აუცილებლად ცალკე კომპონენტი უნდა იყოს).

3) 
  შექმენი პოსტ სერვისი (აუცილებლად სერვისების ფოლდერში ng g s service/posts);

    return [

      },
    ]
  }

4) 
  4.1 home კომპონენტში ჩააინჯექტე პოსტ სერვისი.
  4.2 home კომპონენტში შექმენი ცვლადი Posts: Post[] (ამისთვის შევქმნი პოსტის მოდელს მოდელების ფოლდერში);
  4.3 home კომპონენტში დააიმპემენტირე ngOnInit და ამ მეთოდში კომპონენტის Posts გაუტოლე პოსტსერვისის getposts გაუტოლე;
  4.4 home კომპონენტის html ფაილში *ngFor ით გამოიტანე ყველა პოსტი. გამოიყენე ბუთსთრეფის ქარდი.
